# The Modest Wedding (TMW)

A full-stack wedding booking platform featuring a 14-step booking flow with integrated payments.

## Tech Stack
- Frontend: React + TypeScript
- Backend: Node.js + Express
- Database: currently: PostgreSQL, planned: SupaBase
- Payments: Planned: Stripe, Affirm, ACH, E-Check

## Features
- 14-step booking workflow
- Real-time availability checking
- Multi-payment method support (future)
- Stripe integration (future)
