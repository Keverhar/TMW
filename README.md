# The Modest Wedding (TMW)

A full-stack wedding booking platform featuring a 14-step booking flow with integrated payments.

## Tech Stack
- Frontend: React + TypeScript
- Backend: Node.js + Express
- Database: PostgreSQL
- Payments: Stripe

## Features
- 14-step booking workflow
- Real-time availability checking
- Multi-payment method support
- Stripe integration
